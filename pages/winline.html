WinLine

The brand-new, lightweight terminal for Windows and Linux


What is WinLine?

WinLine is a new, lightweight terminal packed with powerful features to maximize productivity. With constant updates and bug fixes, this terminal strives to provide the best and most stable user experience possible.


Key Features


  • Powerful built-in commands: hard-coded in WinLine itself. These commands behave the same no matter the platform
  • Smooth system intergration: with minimal configuration, WinLine can run system-provided commands like they were built-in
  • Works offline: even without a network connection, almost all features will continue to work without inturruption
  • Cross-platform: supported on many platforms and operating systems, including Windows and Linux
  • Custom commands: Write your own commands or scripts, then install and run them in WinLine
  • Native intergration with Powershell, Command Line, and the Linux terminal

    • Minimum Requirements


  • Python 3
  • 110 KB free space for core files
  • 10 KB free space for documentation
  • Windows 10+, Linux
  • 4 GB RAM (Python 3 requirement)
